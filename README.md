# 🛡️ Project Aegis - Private AI Assistant

A powerful, privacy-focused AI assistant built with advanced security features, biometric authentication, and intelligent conversation capabilities. Project Aegis provides a secure environment where you can interact with an AI assistant while maintaining complete control over your data, permissions, and multi-user access.

## 🌟 Overview

**Project Aegis** is a sophisticated private AI assistant designed for secure, personalized interactions. With integrated biometric security, intelligent memory management, and granular permission controls, Aegis puts you in complete control of your AI experience while ensuring maximum privacy and security.

### Core Philosophy
- **Privacy First** - Your data stays private and secure
- **User Control** - Complete control over permissions and data
- **Security-Centric** - Advanced biometric and authentication features
- **Personalized** - AI learns your preferences and context
- **Multi-User Safe** - Managed user access with role-based controls

## 🚀 Getting Started

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/SharvilMishra/Aegis.git
   cd Aegis
   ```

2. **Setup Environment**
   - Ensure Node.js is installed (if using backend)
   - Configure API keys for AI services
   - Set up authentication credentials

3. **Install Dependencies**
   ```bash
   npm install
   # or
   pip install -r requirements.txt
   ```

4. **Configure Settings**
   - Set up Google OAuth credentials
   - Configure biometric API keys
   - Initialize database connections

5. **Run Locally**
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npm start
   
   # Using Live Server
   npm run dev
   ```

6. **Access the Application**
   - Open `http://localhost:8000` in your browser
   - Or visit: [Project Aegis](https://sharvilmishra.github.io/Aegis/)

### Initial Setup

1. **Login with Gmail**
   - Click "Continue with Gmail"
   - Authenticate your Google account

2. **Biometric Setup**
   - **Face Recognition** - Scan your face for identification
   - **Voice Recognition** - Record voice sample for verification

3. **Permission Configuration**
   - Grant or deny access to specific features
   - Configure file system and device access
   - Set language and response preferences

4. **Start Using Aegis**
   - Create your first conversation
   - Set up user preferences
   - Configure memory options
